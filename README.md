# Transformation Roadmap Generator

AI-powered transformation roadmaps across ERP/SAP, CRM/Salesforce, and Tech Enablement — tailored by sector, focus, and maturity level.

## Features
- 🔷 **ERP / SAP Roadmap** — S/4HANA migration, finance transformation, phased implementation
- 🔶 **CRM / Salesforce Roadmap** — Sales, Service & Marketing Cloud adoption plan
- 🔹 **Tech Enablement Roadmap** — Cloud, AI/ML, data platform, integration architecture
- 📄 **PDF Download** — Professional combined report covering all 3 modules
- 📝 **Markdown Download** — Raw content for further editing

## Sectors
- 🏦 Banking & Financial Services
- 🏥 Healthcare
- 🛒 Retail & FMCG
- ⚡ Energy & Utilities
- 🏛️ Public Sector
- ⚙️ Manufacturing & Industrials

## Setup
Add `GROQ_API_KEY` to Space secrets.

## Tech Stack
Python · Streamlit · Groq (free LLM) · FPDF2 · Docker
